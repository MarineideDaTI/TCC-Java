@startuml
ator Cliente
participant "Interface do Sistema" as Sistema
participant Cliente
participant Evento
participant Ingresso
participant Pagamento

note right of Cliente: Escolhe evento e ingresso\npara comprar

Client -> Sistema: Seleciona evento e ingresso desejados
Sistema -> Sistema: Valida disponibilidade\n e preço do ingresso
Sistema -> Cliente: Mostra detalhes do evento\n e opções de ingresso
Cliente -> Sistema: Confirma seleção de ingresso

loop Pagamento
    Cliente -> Sistema: Seleciona método de pagamento
    Sistema -> Sistema: Processa pagamento
    Sistema -> Cliente: Confirmação de pagamento bem-sucedido
end

Sistema -> Ingresso: Gera ingresso para cliente
Sistema -> Cliente: Entrega ingresso digital

@enduml
